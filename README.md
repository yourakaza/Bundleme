# Bundleme
Make a Bundle from your projects easily.

## Download
You can download it from the [Releases page.](https://github.com/yayydev/Bundleme/releases)

## Usage
After downloading, you can use it like these:
```
bundleme bundle --path "C:\myproject"
bundleme bundle
```
